Hollywood-style representation for Edyta Śliwińska

CAA desk model · portfolio.dgacapital.com/sliw/

This document explains what lives “below the line”: the desk model, engagement strategy, how Edyta uses the app, the Lead Engine, outreach rules, Gamma packaging, weekly cadence, and the parallel Wedding Agent product.

1. What this is

Sliw Agent is a CAA / William Morris–style representation desk. It researches corporations (and separately, wedding clients), matches Edyta’s packages, builds Gamma marketing decks, drafts outreach for human approval, and filters warm leads so Edyta only spends time on conversations that can book.

4. Architecture below the line

Lead Engine → Package + Gamma → Outreach drafts → Human approve → Replies filtered → Edyta briefs only.

Stages: research → scored → packaged → drafted → approved → contacted → replied → interested → discovery_booked → won | lost | nurture

  • Never auto-send
  • Never invent pricing/logos/testimonials
  • One package per first email
  • Corporate and Wedding books stay separate

5. How Edyta uses the app

  • Open Edyta’s desk (warm leads only)
  • Read one-page brief → 15-min discovery → confirm package/date/headcount
  • Stage → discovery_booked / won / nurture
  • Deliver on the floor; desk handles admin

She does not: cold research, chase vague maybes, commodity discounting.

Desk owns: Lead Engine, Gamma, approve sends, sequences, qualify replies.
